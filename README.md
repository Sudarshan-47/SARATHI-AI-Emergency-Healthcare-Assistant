# 🏥 SARATHI AI – Emergency Healthcare Assistant

SARATHI AI is an **AI-powered emergency healthcare assistant** designed to provide quick first-aid guidance, symptom triage, and multilingual voice-based assistance during emergency situations.

## ✨ Features

* 🎤 Real-time **voice input (mic support)**
* 🗣️ Multilingual support: **English, Hindi, Telugu**
* 🚨 Emergency severity classification:

  * Low
  * Medium
  * High
  * Critical
* 🩺 First-aid guidance suggestions
* 🏥 Nearest hospital recommendations
* 🤖 NLP-based symptom analysis
* 📱 Mobile-friendly interface
* 🔊 Human-like voice responses

## 🧠 Problem Statement

In emergency situations, users often panic and need immediate guidance before professional medical help arrives.

SARATHI AI helps by:

* identifying symptoms quickly
* classifying severity
* suggesting first-aid actions
* guiding users in their local language

## 🛠️ Tech Stack

* Replit
* TypeScript / JavaScript
* NLP
* Voice Input APIs
* Web / Mobile UI
